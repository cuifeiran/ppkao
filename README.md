# ppkao
考试资料网显示答案
## 安装使用
1、请百度自行安装油猴（tampermonkey）插件，这款神器就不多BB了，并且插件有中文语言的，莫担心。

2、then，点击这里：[考试资料网跳转显示答案](https://greasyfork.org/zh-CN/scripts/375373-考试资料网跳转显示答案)安装脚本

3、打开考试资源网，你会发现已经可以自动跳转显示答案了
